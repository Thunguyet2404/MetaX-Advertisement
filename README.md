# MetaX-Advertisement

Metax là ứng dụng hỗ trợ kiểm tra và phân tích tài khoản quảng cáo Facebook nhanh chóng.  
Công cụ cho phép kiểm tra Ads Account, Page, Business Manager (BM), Campaign và các dữ liệu liên quan.

## Với Metax, bạn có thể:

- Kiểm tra trạng thái tài khoản quảng cáo (**Ads Account**)
- Xem danh sách **Page** liên kết với tài khoản
- Kiểm tra **Business Manager (BM)** đang quản lý
- Xem thông tin **Campaign** và hoạt động quảng cáo
- Phân tích nhanh tình trạng tài khoản để phục vụ quản lý hoặc marketing

---

Dự án được **Hiếu Trung Nguyễn** chia sẻ dưới dạng **mã nguồn mở**, nhằm hỗ trợ cộng đồng học tập, nghiên cứu và phát triển thêm các tính năng mới.
