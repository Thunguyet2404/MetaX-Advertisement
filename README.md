# MetaX-Advertisement
Metax là ứng dụng hỗ trợ kiểm tra và phân tích tài khoản quảng cáo Facebook nhanh chóng. Công cụ cho phép kiểm tra Ads Account, Page, Business Manager (BM), Campaign và các dữ liệu liên quan. Dự án được Hiếu Trung Nguyễn chia sẻ mã nguồn mở nhằm hỗ trợ cộng đồng học tập, nghiên cứu và phát triển thêm.
